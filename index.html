Jakub Jachimek

| Inżynier | Pasjonat programowania i baz danych |

O mnie

„Jestem absolwentem Politechniki Gdańskiej, gdzie ukończyłem studia na kierunku Zarządzanie Inżynierskie, specjalizacja Technologie Informatyczne. Podczas studiów realizowałem projekt inżynierski polegający na stworzeniu funkcjonalnej bazy danych przy użyciu SQL Server Management Studio. Moje zainteresowanie technologiami komputerowymi rozwinęło się głównie w obszarze baz danych, programowania oraz tworzenia aplikacji webowych. Obecnie realizuję kurs na platformie freeCodeCamp, gdzie uczę się budować aplikacje webowe. W międzyczasie pracuję nad własną aplikacją, aby jeszcze bardziej rozwijać swoje umiejętności w tworzeniu oprogramowania.”

Umiejętności

  • Bazy danych: SQL, SQL Server Management Studio, projektowanie baz danych, optymalizacja zapytań SQL, praca z bazami danych.
  • Programowanie: Java, C++, Python – podstawy programowania obiektowego, algorytmy, struktury danych.
  • Web development: HTML, CSS, JavaScript, React – tworzenie responsywnych stron internetowych, interaktywność aplikacji.
  • Narzędzia: Git, GitHub – wersjonowanie kodu, Docker – konteneryzacja aplikacji, SQL Server Management Studio.
  • Metodologie: Agile, Scrum – zarządzanie projektami w metodyce Agile, organizacja pracy w zespole developerskim.
  • Analiza danych: Excel – zaawansowane funkcje i formuły, Power BI – tworzenie raportów i wizualizacji danych, SAP – obsługa systemów ERP w zakresie analizy danych.”
